# Semester-Tracker

A JavaFX-based semester management application that allows students to organize courses, assignments, exams, and grades through a graphical user interface.

The application helps users track deadlines, monitor academic performance, and analyze workload distribution throughout the semester.

---

## Overview

The Semester Tracker enables users to:

- Add and manage courses  
- Add assignments and exams for each course  
- Record and update grades  
- Update assignment status (complete / not complete)  
- View all deadlines for a course  
- View grades for a specific course  
- Identify the course with the most or least tasks  
- Receive recommendations on which course to focus on based on performance

The system provides structured data visualization through a user-friendly JavaFX interface.

---

## Data Storage

The application stores:

- Course names and course numbers  
- Assignments (name, due date, status, weight)  
- Exams (name, date, status, weight)  
- Grades  
- Deadline information  

Data can be:

- **Saved to a CSV file**
- **Loaded from a CSV file**

This allows users to persist their data between sessions.

## User Interface

The application is built using **JavaFX**, providing:

- Interactive menu navigation  
- Structured input forms  
- Dynamic display of course and deadline data  
- Organized layout for managing academic information  

The GUI improves usability compared to a command-line interface and provides a cleaner academic tracking experience.
